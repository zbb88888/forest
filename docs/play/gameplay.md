# Dark Forest 游戏玩法指南

## 游戏概述

《黑暗森林》是一款2D像素风自动化生产塔防游戏，玩家扮演"最后的管理员"，在废土世界中通过种植植物、指挥机器人、收集装备来重建文明，对抗AI机械势力的威胁。

## 核心玩法机制

### 1. 环境系统

#### 地形类型
游戏世界包含6种地形，每种地形都有独特的特性：

- **草地**（绿色）：基础地形，可种植草类植物，能源产出倍率 1.0x
- **森林**（深绿色）：可种植灌木，能源产出倍率 1.5x
- **山脉**（灰色）：不可通行，能源产出倍率 0.5x
- **水域**（蓝色）：不可通行，能源产出倍率 0.8x
- **沙漠**（沙黄色）：可种植耐旱植物，能源产出倍率 0.7x
- **黑暗森林**（深暗绿色）：危险区域，可种植能源花，能源产出倍率 2.0x

#### 时间系统
游戏采用15日月相循环系统，每个周期包含：

**昼夜循环**
- **黎明**（5-7点）：光照逐渐增强，植物生长速度正常
- **白天**（7-18点）：光照最强，植物生长速度 1.5x
- **黄昏**（18-20点）：光照逐渐减弱，植物生长速度正常
- **夜晚**（20-5点）：光照最弱，植物生长速度 0.5x

**月相周期**（15日）
- **第1日 - 新月**：资源产出 2.0x，AI扫描强度 0.1
- **第2-3日 - 峨眉月**：资源产出 1.8x，AI扫描强度 0.2
- **第4日 - 上弦月**：资源产出 1.6x，AI扫描强度 0.3
- **第5-7日 - 盈凸月**：资源产出 1.4x，AI扫描强度 0.4
- **第8日 - 满月**：资源产出 2.0x（平安夜），AI系统过热停机（扫描强度 0.0）
- **第9-11日 - 亏凸月**：资源产出 1.2x，AI扫描强度 0.5
- **第12日 - 下弦月**：资源产出 1.0x，AI扫描强度 0.6
- **第13-14日 - 残月**：资源产出 0.8x，AI扫描强度 0.7
- **第15日 - 朔月**：资源产出 0.5x，AI扫描强度 0.5（终极挑战）

### 2. 植物系统

#### 植物类型
玩家可以种植5种植物，每种植物都有独特的特性：

- **草**：生长速度快，基础能源产出
- **灌木**：生长速度中等，能源产出较高
- **树木**：生长速度慢，能源产出高
- **花朵**：生长速度中等，能源产出中等
- **能源花**：只能在黑暗森林种植，能源产出最高（3.0x）

#### 种植机制
- **操作方式**：右键点击地图上的可种植区域
- **地形要求**：只能在草地、森林或黑暗森林上种植
- **自动选择**：系统会根据地形自动选择合适的植物类型
  - 草地 → 草
  - 森林 → 灌木
  - 黑暗森林 → 能源花

#### 生长系统
- 植物有5个生长阶段，从幼苗到成熟
- 生长速度受昼夜影响：白天生长更快，夜晚生长变慢
- 植物需要水分和营养维持健康
- 成熟的植物可以收获获得能源

#### 收获机制
- **操作方式**：左键点击已成熟的植物
- **收获奖励**：根据植物类型、健康度和成熟度计算
- **能源获取**：收获后能源直接加入玩家背包

### 3. 自动化机器人系统

#### 机器人类型
玩家可以部署4种机器人，每种机器人都有特定的功能：

- **采集机器人**（棕色）：自动采集成熟植物，移动速度中等
- **建造机器人**（绿色）：负责建造和维修，移动速度较慢
- **防御机器人**（红色）：保护基地和植物，移动速度较快
- **侦察机器人**（蓝色）：探索地图，移动速度最快

#### 机器人AI
机器人具有智能AI，可以：
- 自动寻找最近的成熟植物进行采集
- 自动巡逻指定区域
- 能量不足时自动返回基地充能
- 背包满载时自动返回基地卸载资源

#### 能量管理
- 每种机器人都有能量消耗速率
- 机器人执行任务会消耗能量
- 能量耗尽时机器人停止工作
- 返回基地可以充能

#### 生成方式
- **键盘 1**：生成采集机器人
- **键盘 2**：生成建造机器人
- **键盘 3**：生成防御机器人
- **键盘 4**：生成侦察机器人

### 4. 装备和武器系统

#### 装备类型
游戏包含10种装备，分为3个槽位：

**武器**
- 激光枪：基础武器，平衡的伤害和攻击速度
- 等离子炮：高伤害，攻击速度较慢
- EMP爆破枪：中等伤害，对机械敌人有特殊效果
- 轨道炮：极高伤害，攻击速度很慢

**护甲**
- 轻型护甲：基础防御
- 重型护甲：高防御
- 能量护盾：防御+能量恢复

**饰品**
- 太阳能板：增加能量恢复速率
- 电池包：增加能量上限
- 瞄准系统：增加暴击率

#### 装备稀有度
装备有5种稀有度，影响属性倍率：

- **普通**（白色）：1.0x 倍率，60% 概率
- **稀有**（绿色）：1.2x 倍率，25% 概率
- **史诗**（蓝色）：1.5x 倍率，10% 概率
- **传说**（紫色）：2.0x 倍率，4% 概率
- **神话**（金色）：3.0x 倍率，1% 概率

#### 装备属性
每件装备都有以下属性：
- **伤害**：影响攻击造成的伤害
- **攻击速度**：影响攻击频率
- **防御**：减少受到的伤害
- **能量加成**：增加能量恢复或上限
- **暴击率**：造成暴击的概率
- **暴击倍率**：暴击时的伤害倍率

#### 装备操作
- **键盘 E**：在玩家位置生成随机装备
- **键盘 F**：拾取附近的装备
- **键盘 U**：升级当前装备（消耗 50 能源）

#### 装备升级
- 装备可以升级到10级
- 每次升级提升装备属性
- 升级需要消耗能源（50点/次）
- 升级倍率：1.0 + (等级 × 0.1)

### 5. 资源系统

#### 资源类型
游戏中有3种主要资源：

- **能源**：核心资源，用于种植、升级、充能
- **金属**：建造和维修材料
- **土壤**：种植和培育材料

#### 资源获取
- **植物收获**：主要获取能源
- **机器人采集**：自动采集资源
- **地图探索**：发现资源点

#### 资源消耗
- **种植植物**：消耗能源
- **升级装备**：消耗能源
- **机器人充能**：消耗能源
- **建造建筑**：消耗金属和土壤

## 游戏操作指南

### 基础操作
- **WASD**：移动玩家
- **鼠标左键**：收获植物 / 拾取装备
- **鼠标右键**：种植植物
- **ESC**：暂停游戏

### 机器人控制
- **键盘 1**：生成采集机器人
- **键盘 2**：生成建造机器人
- **键盘 3**：生成防御机器人
- **键盘 4**：生成侦察机器人

### 装备系统
- **键盘 E**：生成随机装备
- **键盘 F**：拾取附近的装备
- **键盘 U**：升级当前装备（消耗 50 能源）

### 界面说明
- **左上角**：资源显示（能源、金属、土壤）
- **右上角**：时间显示（天数、时间、昼夜阶段、月相）

## 游戏策略

### 早期策略（第1-7日）
1. 快速种植植物，建立能源基础
2. 生成采集机器人，自动化资源收集
3. 探索地图，寻找资源点
4. 收集基础装备，提升战斗力

### 中期策略（第8-14日）
1. 利用满月（第8日）的高资源产出，快速扩张
2. 部署多种机器人，建立自动化生产线
3. 升级装备，提升战斗力
4. 建造防御设施，准备应对后期挑战

### 后期策略（第15日）
1. 应对朔月的终极挑战
2. 合理使用能源过载机制
3. 协调机器人防御和采集
4. 目标是存活并完成15日周期

## 注意事项

1. **时间管理**：充分利用白天的高生长速度和满月的资源加成
2. **能量管理**：合理分配能源，避免关键时刻能量不足
3. **机器人配置**：根据需要部署不同类型的机器人
4. **装备选择**：根据游戏阶段选择合适的装备
5. **地形利用**：优先在高产出地形种植植物

## 游戏目标

- **短期目标**：建立稳定的资源生产系统
- **中期目标**：升级装备和机器人，提升战斗力
- **长期目标**：存活15日周期，完成游戏
- **终极目标**：在更高难度下完成15日周期，解锁更多内容
