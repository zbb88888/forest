# Dark Forest 开发计划

## 开发优先级

### 1. 环境相关基础逻辑
- [x] 基础地图系统（已完成）
- [x] 完善地图系统
  - [x] 添加更多地形类型（草地、森林、山脉、水域、沙漠、黑暗森林）
  - [x] 实现地图探索机制
  - [x] 实现战争迷雾
- [x] 时间系统
  - [x] 昼夜循环（黎明、白天、黄昏、夜晚）
  - [x] 15日月相系统（新月、峨眉月、上弦月、盈凸月、满月、亏凸月、下弦月、残月、朔月）
  - [x] 时间对游戏的影响（光照变化、资源倍率、AI扫描强度）

### 2. 植物系统
- [x] 植物类型定义（草、灌木、树木、花朵、能源花）
- [x] 植物种植机制（右键种植，根据地形自动选择）
- [x] 植物生长系统（受昼夜影响，白天生长更快）
- [x] 植物收获系统（左键收获，获得能源）
- [x] 植物对环境的影响（地形能源产出倍率）

### 3. 自动化机器人系统
- [x] 机器人类型定义（采集、建造、防御、侦察）
- [x] 机器人AI逻辑（自动寻路、任务分配）
- [x] 机器人任务系统（空闲、采集、建造、巡逻、维修、返回基地）
- [x] 机器人采集系统（自动采集成熟植物）
- [x] 机器人建造系统（基础框架）

### 4. 装备和武器系统
- [x] 装备类型定义（武器、护甲、饰品）
- [x] 武器系统（激光枪、等离子炮、EMP爆破枪、轨道炮）
- [x] 装备属性系统（伤害、攻击速度、防御、能量加成、暴击率、暴击倍率）
- [x] 装备升级系统（消耗能源升级装备，提升属性）
- [x] 装备稀有度系统（普通、稀有、史诗、传说、神话）

## 当前状态
✅ 所有计划的基础功能已完成！
- 环境相关基础逻辑：地图系统、时间系统（昼夜循环和月相）
- 植物系统：种植、生长、收获
- 自动化机器人系统：采集、巡逻、AI逻辑
- 装备和武器系统：装备生成、拾取、升级

## 游戏操作说明
- 右键：种植植物
- 左键：收获成熟植物 / 拾取装备
- 键盘 1-4：生成不同类型的机器人
- 键盘 E：生成随机装备
- 键盘 F：拾取附近的装备
- 键盘 U：升级装备（消耗 50 能源）
